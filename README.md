# XFormat
这是一个简单的c++的字符串格式化工具
```c++

// 使用 '{ + 数字 + }' 形式作为占位符
// 数字代表后面参数的位置, 位置从0开始计算, 比如{0} 表示第一个参数
// 当数字在参数列表中没有匹配的参数时, 选取第1个参数
// 如果要输出 '{' , 使用 "{{"
// 输出单个'}', 可以使用'}'或'}}', 如果是多个连在一起, 每个'}'都要使用'}}', 比如'}}', 用'}}}}' 

std::string value = "10086"
std::string str = x::Format("this value is {0}", value);
```
